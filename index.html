Установка приложения

Добавление ключа

⚡ Добавить в один клик

Скопируйте ссылку, которую вам прислали. Откройте приложение — оно само предложит добавить ключ из буфера обмена.

3

Настройка и Запуск

1. Выберите конфиг reality-first (если не работает — xhttp).

2. Включите Авто-обновление в настройках (Settings -> Advanced -> Subscription).

3. Нажмите большую кнопку Start.

При первом запуске нажмите "Allow", чтобы разрешить VPN.

Настройка
